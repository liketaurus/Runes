# 🔮 Runes

Невеличкі експерименти з рунами — як доповнення до короткої історії  
**«Ідеальний сендвіч»** на Booknet.

---

## ✨ Про проєкт

**Runes** — це невеликий вебексперимент із рунами та геомантичними інтерпретаціями імен/слів.

Що вміє:

- показує рунічні символи для введеного тексту;
- відображає короткі значення рун;
- рахує умовний енергобаланс (**Сила / Розум / Захист**);
- генерує випадковий **прогноз дня**.

## 🚀 Швидкий старт

1. Клонувати репозиторій:

```bash
git clone https://github.com/liketaurus/Runes.git
cd Runes
```

2. Відкрити `index.html` у браузері  
   **або** запустити локальний сервер:

```bash
python -m http.server 8080
```

3. Перейти за адресою:  
`http://localhost:8080`

---

## 🧪 Як користуватися

1. Введи ім’я або слово українською.
2. Натисни **«АНАЛІЗУВАТИ»**.
3. Переглянь рунічні картки та показники.
4. Натисни **«ОТРИМАТИ ПРОГНОЗ ДНЯ»** для випадкової руни дня.
